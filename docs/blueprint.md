# **App Name**: DarkStore e-commerce

## Core Features:

- Product Catalog: Display a dynamic product listing fetched from Javascript mock data.
- Persistent Cart: Maintain a shopping cart state using localStorage for persistence.
- Product Search: Enable product discovery through a search bar.
- Product Filters: Implement category, price, and brand filtering on the product listing.
- Login Form: Include basic login form to access merchant dashboard.
- Merchant Dashboard: Merchant dashboard: manage, add, edit or remove products. Table visualization for sales report.
- Mobile Responsiveness: Mobile-first responsive design using Tailwind CSS.

## Style Guidelines:

- Primary color: Deep red (#941B0C), for branding and action elements.
- Background color: Dark gray (#242526), providing a modern, dark theme.
- Accent color: Light red (#D9534F), for highlighting and interactive elements.
- Headline font: 'Space Grotesk' sans-serif for titles and headings, conveying a modern aesthetic. Body text: 'Inter' sans-serif for main text, offering a neutral, readable appearance.
- Use minimalistic, geometric icons in white or light red to complement the dark theme.
- Employ a grid-based layout for product listings and dashboard components to maintain a structured and clean interface.
- Incorporate subtle transition animations on button hovers and page transitions to enhance user experience without being distracting.
